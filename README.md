# QR Code Card

A simple responsive QR Code card built using HTML and CSS. This project focuses on translating a Figma design mockup into HTML and CSS.

---

## Overview

This project recreates a QR code card component from a design specification from Frontend Mentor. The goal is to practice structuring HTML and styling layouts with CSS while maintaining visual accuracy to the provided design.

---

## Objectives

- Practice translating a design into HTML and CSS
- Improve design skills
- Build a responsive card component
- Gain experience working with real-world design specifications

---

## Built With

- HTML5
- CSS
- Flexbox

---


## What I Learned

While building this project, I practiced:

- Structuring HTML components
- Using Flexbox for layout alignment
